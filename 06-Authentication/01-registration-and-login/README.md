# Registration & Login
* ExpressJS with SQLite

### Third Party Packages
* `npm install bcrypt` create hashed password
* `npm install jsonwebtoken`  Generate JWT token

### Installation
* `npm install`

### Start the Application
* `npm run dev`

### Folder Structure
* src
   - index.js
* rest-client
   - app.http
* database
  - mydb.db
